# typescript-exercise

simple project to learn typescript following this tutorial on Youtube:

https://youtube.com/playlist?list=PL4cUxeGkcC9gNhFQgS4edYLqP7LkZcFMN&si=oAtteb7-1hK5EEyA

### `tsc index.ts`

to compile typescript

### `node index.js`

to run js

### `tsc --init`

create config for typescript

### `tsc --watch`

Auto compile

### `node --watch dist/index.js`

auto run js
